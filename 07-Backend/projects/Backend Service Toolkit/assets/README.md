# Assets — Backend Service Toolkit

## Purpose

Static artifacts supporting documentation and demos: diagrams exported from Mermaid, OpenAPI spec snapshots for reviews, and architecture sketches for portfolio presentations.

## Conventions

| Asset type | Location | Notes |
| --- | --- | --- |
| Mermaid source | Lives in markdown notes | Export PNG/SVG here when needed for slides |
| OpenAPI canonical | [[07-Backend/code/openapi/demo-api.yaml\|07-Backend/code/openapi]] | Docs reference code tree as source of truth |
| Diagram exports | `assets/diagrams/` | Name as `{topic}-{date}.svg` |
| Screenshots | `assets/screenshots/` | Redact secrets and tokens |

## Current Inventory

No binary assets committed yet—documentation embeds Mermaid inline. Add exports when Career portfolio or presentations require static images.
