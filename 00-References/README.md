# 00 References

Curated sources that support curriculum notes. References are not a substitute for first-principles explanations inside topic notes.

## How to Add a Reference

1. Copy [[00-Templates/Reference Template|Reference Template]].
2. Create one note per major source when the source is reused across topics.
3. Link the reference from topic notes under **Further Reading**.
4. Prefer primary sources, classic textbooks, RFCs, and high-signal production engineering writing.
